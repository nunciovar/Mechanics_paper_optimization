# Output Format

## Diagnosis Tasks

Return:

1. Structural problems
2. Logic gaps
3. Claim-evidence risks
4. Repetition hotspots
5. AI-style or template-style hotspots
6. Must-keep scientific content
7. Author-confirmation items

## Rewrite Tasks

Return:

1. Revised text
2. Change notes
3. Open questions
4. High-risk lines not auto-changed

## Reviewer-Style Tasks

Return:

1. Review setup
2. Reviewer-specific assessments
3. Cross-review synthesis
4. Unsupported or not-assessable claims

Use concise Markdown. Do not hide uncertainty inside polished prose.
