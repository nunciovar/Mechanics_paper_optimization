# Terminology Ledger Rules

Before major rewriting, lock the canonical terms that should not drift.

## What To Record

- Material labels and phase names.
- Model names and abbreviations.
- Symbols, units, and local meanings.
- Acronyms and first-use long forms.
- Damage, fracture, debonding, and failure terms.
- Terms that may have multiple plausible technical meanings.

## How To Use

1. Check `style/terminology.yml`.
2. If the manuscript uses additional project-specific terms, add them to a review ledger before broad editing.
3. If a term is ambiguous, mark it as `AUTHOR_INPUT_NEEDED` instead of normalizing it.
4. After rewriting, re-run terminology checks when feasible.
